How to view live event data

Mixpanel event data is only available on the native app (iOS/Android). The web version does not track events.

Once you install the app on a device and generate a plan, events will appear in Mixpanel within seconds.

To view live events:

  1. Click Open Mixpanel Dashboard above
  2. Go to Live View in the left sidebar
  3. Open the native app on a device
  4. Events appear in real time as you use the app

For historical reports, go to Insights and build charts using the event names listed on the left.
