How to view live events

Mixpanel only tracks events on the native app (iOS/Android). Web version is excluded by design.

To see live data:

  1. Click Open Mixpanel above
  2. Go to Live View in the sidebar
  3. Open the native app on a device
  4. Events appear in real time

For historical charts go to Insights and select any event name from the list above.
