How to view live events

Mixpanel tracks events on the native iOS/Android app only. Web version is excluded by design.

Once the app is on the App Store and users install it:

  1. Click Open Mixpanel above
  2. Go to Live View in the left sidebar
  3. Events appear in real time as users use the app
  4. Go to Insights for historical charts
